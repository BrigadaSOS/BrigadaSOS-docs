# ¡Bienvenidos a la Brigada SOS!

![main-haruhi](../assets/introduction/main_haruhi.gif)

:::danger Antes de empezar
Esta no es la única guía que existe o existirá para aprender japonés con un método diferente al tradicional. Esta guía constituye un intento para llevar este método a más personas hispanohablantes que se encuentran limitadas por la barrera del idioma, a un conocimiento accesible para una comunidad mayormente olvidada como la nuestra. 
:::

Si quiere aprender japonés de la forma apropiada o si tiene problemas con el método tradicional como lo son las clases, los libros de ayuda o tutores y quiere alcanzar un nivel nativo o cercano a ello de forma eficaz y divertida, esta guía le será de ayuda.

Aquí no se enseña japonés, **se enseña a aprender japonés.** ようこそ！
