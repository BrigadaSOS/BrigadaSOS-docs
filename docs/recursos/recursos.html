Recursos

Bienvenidos a la sección de recursos de la Brigada SOS. Esta sección está dividida en cuatro categorías para facilitar su busqueda: Aprendizaje, Inmersión, Herramientas y Misceláneo.

Información

La mayoría de recursos disponibles en esta página están en inglés. Con todo ello se entiende que el inglés pueda suponer una barrera, pero de ser así considere usar un traductor de calidad como DeepL para seguir avanzando con contenido de calidad.

Aprendizaje#

Kana#

  • DJTKana - Página web para aprender los Kana a través de la repetición de los mismos.
  • Deck Kana - Deck para Anki que contiene los Kana y su pronunciación.

Kanji#

  • Deck Recognition RTK - Deck necesario para aquellos que les cuesta demasiado memorizar kanjis.

Gramática#

  • Cure Dolly - Canal de Youtube que se encarga de explicar el Japonés de la forma más orgánica posible. La mejor guía que existe a día de hoy.
  • Diccionario de gramática - Contiene entradas sobre la gramática Japonesa (no literales).
  • Decks de gramática - Decks de gramática por DoJG divididos por dificultad.
  • Tae Kim - Guía muy popular, muy simple, algo vaga, pero aceptable.

Vocabulario#

  • MIA Tango N5 Deck - Primer deck opcional recomendado para principiantes.
  • MIA Tango N4 Deck - Segundo deck opcional recomendado para principiantes.

Diccionarios#

  • 広辞苑無料検索
  • Jisho
  • Weblio EJJE
  • Eijirou on the WEB
  • Paquete Diccionarios Yomichan (V2)

Guías externas#

  • TheMoeWay - La mejor guía en inglés que hay para aprender Japonés.
  • Refold - Sitio web que abarca de forma detallada cómo aprender cualquier lenguaje a través de la inmersión.
  • AnimeCards - Contiene una guía simple de cómo aprender Japonés pero donde realmente brilla es por su guía de cómo configurar Anki.

Inmersión#

Anime#

  • Abema - Servicio de Streaming (requiere VPN) más popular en Japón. Podrá encontrar desde televisión, peliculas, anime, etc.
  • Amazon Video JP - Servicio de Streaming proporcionado por Amazon, similar a Netflix.
  • AniDL - Sitio web para descargar anime RAW en HD. Sin torrents.
  • AnimeKaizoku - Sitio web similar a AniDL para descargar anime RAW en HD. Sin torrents.
  • AnimeRaw - Sitio con una interfaz obsoleta pero contiene anime anime RAW tanto viejo como reciente.
  • B9Good - Sitio para ver anime a través de Streaming en formato RAW, página utilizada mucho por japoneses.
  • Itazuraneko - Colección de anime RAW para descargar de forma directa.
  • Netflix - Hay contenido japonés disponible en Netflix pero si cuenta con un VPN Japonés tendrá acesso a una cantidad impresionante de contenido exclusivo de Japón.
  • Nyaa - Sitio por excelencia para obtener torrents, en este caso, de anime RAW.

Sincronización de subtítulos#

Recuerde que puede sincronizar los subtítulos de forma manual desde cualquier reproductor. Las herramientas de esta sección son para sincronizar subtítulos de forma automática y siempre existe la posibilidad de que no sean certeras.

  • AutoSubSync
  • AnimeCards
  • Crear tarjetas en Anki con Subs2srs + Morphman
  • Mazos de subs2srs
  • Sentence mining con MPV
  • Tsukuba Web Corpus (busca oraciones de ejemplos para palabras; ideal para sentence mining)

Misceláneo#

Extensiones de navegador#

  • AutoLink Bypasser - Evita links de publicidad molestos de forma automática. Indispensable.
  • uBlock Origin - Evita la publicidad y rastreadores en cualquier sitio web. Indispensable.
  • TamperMonkey - Permite la inserción de scripts en sitios webs.
  • DF Tube - Permite ocultar los comentarios, videos recomendados y otras cosas que puedan ser una distracción en YouTube.
  • Clipboard Inserter - Inserta en la página el contenido copiado; ideal para sentence mining.
  • Subadub - Permite la selección y descarga de subtítulos de Netflix.

Scripts TamperMonkey#

  • Furigana Remover - Elimina en la mayoría de páginas web aquello que contenga Furigana. Requiere TamperMonkey.

  • AnkiConnect - Permite la conexión de Anki con herramientas externas.
  • AwesomeTTS - Permite añadir audios hechos por una maquina neuronal a cualquier palabra.

  • Advanced Browser - Añade un buscador avanzado de tarjetas):

Varios#

  • DeepL - Traductor basado en Machine Learning. El mejor traductor disponible.
  • NicoNico - Plataforma similiar a Youtube pero exclusiva de Japón.
  • YouGlish - Plataforma que busca un video de Youtube donde se use la palabra que hayas colocado.
  • Tokyo Toshokan - Librería que contiene de todo; videojuegos, novelas, mangas, etc.
  • NINJAL-LWP - Permite buscar oraciones con la palabra deseada.
  • Yahoo JP -
  • uNoGS - Permite buscar todo el contenido de Netflix.
  • YTMP3 - Convertir videos de YouTube a mp3; ideal para inmersión pasiva.
  • Kanji Study - Aplicación para Android que permite estudiar kanjis, separarlos por grupo y exportarlos a Anki.
  • Youtube Vanced - Youtube para Android pero sin publicidad y con opción de escuchar en segundo plano.
  • JPDB - Sitio web para visualizar la dificultad de anime, manga, novelas visuales, etc con funcionalides extras para hacer inmersión.
